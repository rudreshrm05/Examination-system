ONLINE EXAMINATION SYSTEM 
------------------------------------------------

This system provides the platform to conduct MCQ examinations.

REQUIREMENTS
--------------------------

1. Net Beans
2. JDK(java development kit)
3. mysql server 5.5
4. mysql connector.

INSTRUCTIONS TO USE
------------------------------------

Make sure your system have all the above mentioned requirements.

1. Dump the database into the system 
   - Open start run cmd.
   - Type command cd C:\Program files\MySql\MySql Server 5.5\bin
   -mysql -u[username] -p[password] -h[hostname] [databasename] <[complete path of the sql file]

2. Connect to the databse.
   -open services in netbeans.
   -Right click on the MySql, click on connect.
   -Drop down the tree MySql, right click on your database and click connect.
   -Drop the tree drivers, look for jdbc/....databasename.., right click and connect.

3. Running the program.
    Run the file IndexES.java via Net beans.
